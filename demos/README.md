# Demos

Standalone, self-contained design demos (open directly in a browser).

- **meridian-dashboard.html** — *Meridian*, a container-terminal port-operations
  control dashboard. A frontend-design showcase of a mature, interactive dashboard:
  time-range control, hover-crosshair area chart, KPI sparklines, a radial berth-
  occupancy gauge, a live berth-schedule Gantt (the signature), status filters, and
  a click-through vessel detail drawer. Vanilla JS + inline SVG, no build step;
  responsive, keyboard-accessible, reduced-motion aware. Fonts load from Google Fonts.
